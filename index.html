Scrum процесс:

Этот проект разрабатывается по методологии Scrum. В репозитории смоделирован полный цикл спринта:

  • Product Backlog: 8 задач в GitHub Issues
  • Sprint Planning: выбрано 5 задач в спринт
  • Daily Standups: отслеживание прогресса
  • Sprint Review & Retrospective: анализ результатов
